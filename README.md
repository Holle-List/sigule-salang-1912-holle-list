The R code file for processing the digitised Sigulei and Salang (NBG
1912) wordlist (with Indonesian and English glosses and part of the
[Holle List for the Barrier Islands Languages of Sumatra,
Indonesia](https://github.com/complexico/holle-list-barrier-islands)) is
available
[here](https://github.com/complexico/holle-list-barrier-islands/blob/main/processing-sigule-and-salang-googlespreadsheet.R)
from [this
repository](https://github.com/complexico/holle-list-barrier-islands).
The digitised version of the Sigulei and Salang (NBG 1912) wordlist is
part of the group project in [this
repository](https://github.com/complexico/lexico-holle-list-barrier-islands).

## How to cite

If you use the data in this repository, please cite the original sources
([Stokhof, 1980](#ref-stokhofHolleListsVocabularies1980); [Stokhof &

> lists, vocabularies in languages of Indonesia, Vol. 1: Introductory

2023](#ref-rajegDigitisedSearchableHolle2023b)), from which the
